# Mini_Translator
Mini translator with explanations for language learners
# Mini-Übersetzer mit Erklärungen für Lernende

Dieses Projekt übersetzt Texte und erklärt grammatische Unterschiede
für Deutschlernende (B1–C1).

## Features
- Englische → Deutsche Übersetzung
- Grammatik-Erklärungen
- Lernorientierter Output

## Tech Stack
- Python
- Streamlit
- Hugging Face Transformers
- spaCy
